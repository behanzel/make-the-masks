This directory contains documents that can/should be included with each mask or delivery of masks. They include:

## Handouts for Each Mask
- A [customizable letter](https://github.com/blackbear/make-the-masks/blob/master/documents/maskletter.docx?raw=true) if you are giving out Filtrete filter paper with each mask.
- A [customizable letter](https://github.com/blackbear/make-the-masks/blob/master/documents/genericletter.docx?raw=true) if you are giving out the oval KN95 filters with each mask.
- A [customizable letter](https://github.com/blackbear/make-the-masks/blob/master/documents/noletter.docx?raw=true) if you are not giving out any kind of filter with each mask.

## Labels for Each Mask

A Microsoft Word template that you can customize with your name, and print on 2"x4" label stock (Avery 5163/8163 or equiv)
- [Small](https://github.com/blackbear/make-the-masks/blob/master/documents/masklabels_small.docx?raw=true)
- [Medium](https://github.com/blackbear/make-the-masks/blob/master/documents/masklabels_medium.docx?raw=true)
- [Medium Large](https://github.com/blackbear/make-the-masks/blob/master/documents/masklabels_ml.docx?raw=true)
- [Large](https://github.com/blackbear/make-the-masks/blob/master/documents/masklabels_large.docx?raw=true)
- [XL](https://github.com/blackbear/make-the-masks/blob/master/documents/masklabels_xl.docx?raw=true)
- [XXL](https://github.com/blackbear/make-the-masks/blob/master/documents/masklabels_xxl.docx?raw=true)
